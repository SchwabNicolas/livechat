# Installation

-   Cloner la repo
-   pip install -r requirements.txt
-   Migrations
-   Createsuperuser
-   Se connecter avec le superuser sur fireox (besoin de firefox pour les multi-containers)
-   Créer un deuxième container et se connecter avec le deuxième compte créé depuis l'admin panel
-   Aller sur localhost:8000/dialogs/Khodok et avec l'autre sur localhost:8000/dialogs/boiboi
